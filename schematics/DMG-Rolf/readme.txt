Info
====

Eagle schematics for the DMG-Rolf project, a project aiming to clone the Gameboy
mother and peripheral boards.

Files
=====

DMG-Rolf-110619.zip
The latest official version from Rolf.

DMG-Rolf-nitromod-101013.zip
Motherboard only, but with some trace work done. In particular, the RAM chip
below the CPU was placed closely to the CPU and hand-routed.
