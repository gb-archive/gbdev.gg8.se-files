This tool can be used to patch your own samples into Little Sound Dj!

Tutorial by Little-Scale: 
http://little-scale.blogspot.com/2008/11/how-to-prepare-samples-and-create-lsdj.html

UPDATES: 

25-Mar-2007: v6

 - added support for loading 16-bit WAV samples
 - added menu option "Import kits from ROM" for importing all kits from
   another ROM image. useful at version upgrades

8-Aug-2006: v5

 - bugfixed. problems with creating new kits on previously unused
 kit spaces

8-Jul-2006: v4 

 - made room for more kits - now possible to fit about 50 kits on each 
   ROM image. this new space will only be visible on LSDj v3.6.0 and up

NOTE: You need Java 1.4 or higher to run this one.
